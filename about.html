Reactor Simulator
Simulator About

I made this project as a tool to help quickly develop Bigger Reactors and Extreme Reactors designs for the game Minecraft. The only other tool I could find was out of date and had a poor selection of moderators, so I made my own. I hope it is simple to use, but if you have suggestions leave them on the github repo and I might implement them. Please report any bugs to the github repo as well and I will fix them as quickly as possible. This is version two, so there are bound to be bugs or improvements. In addition to this being just version two, in the future I hope to expand on this simulator to turbine design and simulation.
